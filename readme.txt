Myriad: 

Project Description: 
Titled Myriad, the single-player game will explore a character — chosen from several options — surviving boss fights and obstacles, obtaining power-ups and collecting items to earn the highest score possible. Bosses and scattered traps will decrease the player's health and once the player loses all their health, the game ends. 

How to run the project:
1. Download all files 
2. Run the tpMain.py file 

Libraries needed: 
1. cmu_graphics

Shortcuts:
All of the screens are accessible using the buttons
